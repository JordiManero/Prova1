Do not save unfinished charts. This page makes use of the GitHub API to store your charts. There’s a rate limit of 60 anonymous API requests per-hour per-visitor. To avoid reaching this limit, this sofware tries to use the caching mechanisms available in modern browsers. If you cannot save your chart, please try later.

To build your chart (particularly, to fill in the time, data, by or grouped fields), you might need to read the documentation or check some sample code. See some visualizations done with Visual Maker.
